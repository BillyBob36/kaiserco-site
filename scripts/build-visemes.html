// BUILD VISEMES // KAISER CO

Cette page lit les MP3 déjà générés dans public/assets/voices/, les analyse via wawa-lipsync, et produit un fichier visemes.json consolidé.

Ouvre cette page via un serveur local (ex: npx http-server public -p 8080 à la racine + ouvrir http://localhost:8080/../scripts/build-visemes.html) ou utilise npm run serve.
